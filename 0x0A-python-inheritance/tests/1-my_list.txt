Simple cases:

>>> from my_module import MyList
>>> my_list = MyList([3, 1, 2])
>>> my_list.print_sorted()
[1, 2, 3]

>>> my_list = MyList([5, 3, 4, 1])
>>> my_list.print_sorted()
[1, 3, 4, 5]

Edge cases:

>>> my_list = MyList([])
>>> my_list.print_sorted()
[]

>>> my_list = MyList([1])
>>> my_list.print_sorted()
[1]

Duplicate values:

>>> my_list = MyList([4, 4, 2, 2])
>>> my_list.print_sorted()
[2, 2, 4, 4]

Negative values:

>>> my_list = MyList([-1, -3, 0, 2, -2])
>>> my_list.print_sorted()
[-3, -2, -1, 0, 2]
